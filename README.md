# w3-oopgame

# How to Play

# Click START GAME to begin.

# Read the room description.

# Choose a direction to move or pick up items.

# Collect all three ship parts:

# Milkshake machine motor

# Goal post netting

# Big Bens little hand

# Return to the Crash Site to win.

# If you enter the Centre for Disease Control without all parts, you lose.
